# Política de Seguridad

## Versiones Soportadas

Actualmente, solo se ofrecen actualizaciones de seguridad para la última versión del proyecto.

| Versión | Soportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reportar una Vulnerabilidad

Si encuentras una vulnerabilidad o un problema de seguridad en este proyecto, por favor **NO abras un Issue público**.

En su lugar, envía un correo electrónico a **jordanmonier4@gmail.com**.

Trataré de responder y solucionar el problema lo antes posible.
